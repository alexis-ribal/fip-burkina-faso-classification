# Satellite-based tree cover mapping for forest conservation in the drylands of Sub Saharan Africa (SSA): Application to Burkina Faso gazetted forests
This repository contains the GEE code used to classify Sentinel-2 images and create a visualization APP

The final product can be accessed https://ariverab.users.earthengine.app/view/forestclassification-burkinafaso-gazettedforests

The code to create this app is the following:
